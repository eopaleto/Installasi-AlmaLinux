# Installasi Alma Linux Pada Virtual Box

1. Download file iso Alma Linux pada link berikut:

```https://repo.almalinux.org/almalinux/9.3/isos/x86_64/AlmaLinux-9.3-x86_64-minimal.iso```

![Screenshot (53)](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/29fca463-89ae-475b-8a27-6f56c0190c68)

2. Setelah di Download, Buka Virtual Box Anda dan klik `New`
   
![1](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/37ea6584-ae64-43e3-8c4c-3f7d04b3db90)

3. Buat nama Operasi System menjadi `Alma Linux`

![2](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/54d81a4a-9c35-4f51-9172-00aeb92ffef0)

4. Kemudian klik panah bawah pada kolom `iso image` dan klik other/lainnya untuk memilih folder yang kalian download tadi

![3](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/cdcf125c-e165-4162-8c22-b6d45f7440d2)

5. Pilih file iso yang sudah kalian Download diatas tadi kemudian klik `Open`

![4](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/69941dd6-4531-4751-8f56-7afd5e71dfc1)

6. Dan jangan lupa centang pada bagian `Skip Unttanded Installation`

![5](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/d1ff7585-aa5c-4ac3-94d5-0d5a9a358cdd)

7. Atur sesuai keinginan dan sesuai dengan komputer/laptop anda kemudian klik `Next`

![6](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/cb2522a1-7d2d-48d2-b6f3-027f5114f83c)

8. Klik `Next`

![7](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/26dad054-d25d-4d64-b50c-805d6f981c7c)

9. Dan Klik `Finish`

![8](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/c289d3d5-a6c4-4ea1-a77d-eab25797a5b5)

10. Nah ketika sudah diinstall maka buka setting, kemudian atur pada bagian `Network`, Ubah Attached to menjadi `Bridged Adapter` dan Pada bagian Advanced bagian Permision ubah menjadi `Allow All`.

![10](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/7dcc7c6d-712c-4a63-ac6c-5c65f1891844)

11. Setelah di setting, Maka Jalankan Alma Linux dengan klik `Start` pada Bagian `AlmaLinux`

![9](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/dd484aa4-2d53-4684-adef-6879f2ac48fb)

12. Kemudian klik `Install AlmaLinux`

![11](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/3036ed60-67e8-4f2f-b537-ea2f155ec074)

13. Tunggu Hingga Installasi Alma Linux Selesai.

14. Setelah masuk, Pilih sesuai `Negara` dan `Bahasa` anda. Dan klik `Lanjutkan`
    
![12](https://github.com/eopaleto/Installasi-AlmaLinux/assets/126212773/149147e0-20d1-4d42-b54f-23de1c2ef364)

15. Kemudian, isi pada bagian `Setelan Pengguna` seperti `Kata Sandi`, `Pembuatan Pengguna`, Setelah itu Klik `Mulai Installasi`.

16. Dan Ketika suda, klik `Mulai Ulang`

---

  **Selamat! Anda sudah berhasil menginstall Alma Linux pada Virtual Box.**
